# Monochrome 0.96" 128x64 OLED Graphic Display - I2C Interface
Sample code for connecting 0.96" Monochrome OLED display to Arduino Pro 3.3V/8MHz using I2C interface

Wiring:</br>
Ardino Pro 3.3V/8MHz ------- OLED Display (I2C)</br>
====================================</br>
GND ----------------------------> GND</br>
3.3V ----------------------------> VCC</br>
A4 or SDA -----------------------> SDA</br>
A5 or SCL -----------------------> SCL</br>
